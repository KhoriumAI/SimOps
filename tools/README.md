# Tools & Utilities

This directory contains shared helper libraries and standalone tools for analysis, benchmarking, and visualization.

## Structure

- **analysis/**: Tools for deep mesh analysis (metrics, validation).
- **benchmarking/**: Scripts for performance testing (CPU/GPU comparison).
- **visualization/**: Utilities for rendering and visual debugging.
- **testing/**: Shared test infrastructure (distinct from the specific tests in `verification_lab`).
