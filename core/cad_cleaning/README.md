# CAD Cleaning

This directory contains specialized algorithms and utilities for cleaning and repairing CAD geometry before meshing.

## Purpose
- Defeaturing (removing small holes, fillets, logos).
- Healing (fixing gaps, overlaps, and self-intersections).
- Sharp edge classification and preservation.
- Surface loop closure for watertightness.
