# Applications

This directory contains the entry points for the different interfaces of the Khorium Mesh Package.

## Structure

- **desktop/**: The main desktop GUI application (`gui_app`).
- **web/**: The Streamlit-based web interface (and potentially other web frontends).
- **cli/**: Command-line interface tools and worker scripts (e.g., `mesh_worker_subprocess.py`).
- **montage_gui.py**: Standalone GUI for creating mesh showcase videos.
