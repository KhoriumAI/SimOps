# Khorium Toolbox

A collection of generalized skills and utilities for use across the project.

## How to contribute
- When you write code that solves a problem you feel can be generalized, add it here.
- Include a comment at the top explaining what it does and how to use it.
- Ensure the code is modular and reusable.

## How to use
- Reference these skills when stuck on a common problem.
- Check here before implementing a common utility to avoid duplication.
